# Basic Chatbot

## Description
This is a simple rule-based chatbot built using Python. The chatbot responds to predefined user inputs like greetings, questions, and exit commands.

## Features
- Greets the user
- Responds to simple questions
- Continuous conversation until the user types "bye"
- Console-based interaction

## Technologies Used
- Python 3

## Concepts Used
- While Loop
- If-Elif-Else Statements
- User Input
- String Operations

## How to Run
1. Install Python 3.
2. Download the project.
3. Run:# CodeAlpha_chatbot
A simple rule-based chatbot built in Python that responds to predefined user inputs using conditional statements.
